# Sistema de Compras

Aplicación con React + Node.js + PostgreSQL

## Estructura
- `app-compras/` - Frontend con React y Material UI
- `backend/` - API REST con Express y JWT

## Instalación
1. Instalar dependencias:
   - `cd backend && npm install`
   - `cd ../app-compras && npm install`

2. Ejecutar backend:
   ```bash
   npm run dev
